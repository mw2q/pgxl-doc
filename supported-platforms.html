15.6. Supported Platforms

Postgres-XL can be expected to work on these operating systems: Linux (all recent distributions), FreeBSD and Mac OS X. Other Unix-like systems may also work but are not currently being tested.

If you have installation problems on a platform that is known to be supported according to recent build farm results, please report it to <postgres-xl-bugs@lists.sourceforge.net>. If you are interested in porting Postgres-XL to a new platform, <postgres-xl-developers@lists.sourceforge.net> is the appropriate place to discuss that.
